Display Property

What is Display?

The display property defines how an element is shown on the page.


Common values

block

Takes full width and starts on a new line.

Example 
div {
  display: block;
}
inline

Does not start on new line.

Example 
span {
  display: inline;
}

inline-block

Inline but allows width and height.

Example 
button {
  display: inline-block;
  width: 150px;
}
None

Hides the element.

Example 
p {
  display: none;
}
  1. Position Property
What is Position?

The position property controls how elements are positioned.

static (default)
Example 
div {
  position: static;
}
relative

Moves relative to its normal position.

Example 
div {
  position: relative;
  top: 20px;
}
absolute

Positions relative to nearest positioned parent

Example 
div {
  position: absolute;
  top: 0;
  right: 0;
}
fixed

Stays fixed even while scrolling

Example 
div {
  position: fixed;
  bottom: 0;
}

Flexbox – Important Properties

Container Properties
  1. display: flex;

Makes the element a flex container.

Example 
.container {
  display: flex;
}

flex-direction

Defines direction of items

Values:
  • row (default)
  • column
  • row-reverse
  • column-reverse
Example 
.container {
  flex-direction: column;
}

justify-content

Aligns items horizontally.

Values:
  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly
Example 
.container {
  justify-content: space-between;
}

align-items

Aligns items vertically

Values:

  1. stretch
  2. center
  3. flex-start
  4. flex-end
  5. baseline
Example 
.container {
  align-items: center;
}

  1. flex-wrap

Controls wrapping.

Values:
  • nowrap
  • wrap
  • wrap-reverse
Example 
.container {
  flex-wrap: wrap;
}

Item properties

  1. flex-grow

Defines how much item grows.

.box {
  flex-grow: 1;
}
  1. flex-shrink
.box {
  flex-shrink: 1;
}
  1. flex-basis

Defines initial size.

Example 
.box {
  flex-basis: 100px;
}

CSS Grid – Important Properties

display: grid;

Example 
.container {

grid-template-columns

Defines column structure.

.container {
  grid-template-columns: 100px 200px 100px;
}

grid-template-rows

Example 
.container {
  grid-template-rows: 100px 100px;
}

gap

Example 
.container {
  gap: 10px;
}

grid-column & grid-row

Example 
.item {
  grid-column: 1 / 3;
