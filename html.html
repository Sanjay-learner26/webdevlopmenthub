Learn HTML

"HTML is where web development begins."

What is HTML?

  1. HTML stands for Hyper Text Markup Language.
  2. It is the standard markup language for creating web pages.
  3. HTML describes the structure of a webpage.
  4. HTML consists of a series of elements.
  5. HTML elements label content such as headings, paragraphs, and links.

Why HTML is Used?

  • Structure of web pages.
  • Works with CSS and JavaScript.
  • Easy to learn.

Basic HTML Page Structure

HTML Example 
<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

Common HTML Tags

Tag Description
<a>Creates a hyperlink
<img>Adds an image
<strong>Important text
<ol>Ordered list
<ul>Unordered list
<li>List item
<em>Emphasis text
<mark>Highlight text
<table>Create table
<form>Create form
<del>Deleted text
