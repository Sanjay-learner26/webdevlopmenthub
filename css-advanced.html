Transition – Important Properties

transition-property

transition-property: background;

transition-duration

Example 
transition-duration: 1s;

transition-timing-function

Values:

  • ease
  • linear
  • ease-in
  • ease-out
  • ease-in-out
Example 
transition-timing-function: ease-in-out;

transition-delay

Example 
transition-delay: 0.5s;

scale()

Example: 
transform: scale(1.2);

rotate()

Example 
transform: rotate(45deg);

translate()

Example 
transform: translate(50px, 0);

skew()

Example 
transform: skew(20deg);

Animation – Important Properties

@keyframes

Example 
@keyframes move {
  from { left: 0; }
  to { left: 200px; }
}

animation-name

Example 
animation-name: move;

animation-duration

Example 
animation-duration: 2s;

animation-iteration-count

Example 
animation-iteration-count: infinite;

animation-direction

Values

  • normal
  • reverse
  • alternate
animation-direction: alternate;

Z-Index

Works only with positioned elements.

.box {
  position: absolute;
  z-index: 10;
}

Higher number = on top.

Values:

  • visible
  • hidden
  • scroll
  • auto
Example
