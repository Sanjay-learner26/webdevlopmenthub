HTML APIs & Advanced

HTML5 introduced powerful APIs for building interactive web applications. These features allow browsers to store data, draw graphics, and access device location. They make modern websites dynamic without needing external plugins.

Geolocation API

The Geolocation API is used to get the userâ€™s current location. It requires user permission before accessing location data. It is commonly used in maps and delivery applications.


Example 
<button onclick="getLocation()">Get Location</button>

<script>
function getLocation() {
  navigator.geolocation.getCurrentPosition(function(position) {
    alert("Latitude: " + position.coords.latitude);
  });
}
</script>

Drag and Drop

Drag and Drop allows users to move elements on a webpage. It improves user interaction and UI experience. It uses the draggable attribute.

Code Example 
<div draggable="true">Drag Me</div>

Local Storage

Local Storage stores data in the browser permanently. Data remains even after closing the browser. It is useful for saving user preferences.

Code Example 
<script>
localStorage.setItem("username", "Sanjay");
console.log(localStorage.getItem("username"));
</script>

Session Storage

Session Storage stores data temporarily. Data is deleted when the browser tab is closed. It is useful for temporary user data.

Code Example 
<script>
sessionStorage.setItem("sessionUser", "Guest");
console.log(sessionStorage.getItem("sessionUser"));
</script>

Tag: <canvas>

The <canvas> tag is used to draw graphics using JavaScript. It is used for games, animations, and charts. Canvas requires JavaScript to create shapes.

Code Example 
<canvas id="myCanvas" width="200" height="100"></canvas>

<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.fillRect(20, 20, 100, 50);
</script>

Tag: <svg>

SVG stands for Scalable Vector Graphics. It is used to create vector-based graphics in HTML. SVG graphics do not lose quality when resized.

Example 
<svg width="200" height="100">
  <circle cx="50" cy="50" r="40" fill="blue" />
</svg>
