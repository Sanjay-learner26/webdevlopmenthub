LOCAL STORAGE

This is where your website starts remembering data.

What is LocalStorage?

LocalStorage is a browser storage system.

It allows you to:

  • Store data in browser
  • Save user input
  • Save cart items
  • Save login state
Important:

Data stays even if you close the browser.

Why It Is Important?

Without LocalStorage:
If user refreshes page → all data disappears.

With LocalStorage:
Data stays saved.
Used in:
  • Todo apps
  • Shopping cart
  • Dark mode toggle
  • Login systems

Basic Methods of LocalStorage

LocalStorage stores data in key–value format.

setItem() → Store Data
JavaScript 
localStorage.setItem("name", "Sanjay");
getItem() → Get Data
JavaScript 
let name = localStorage.getItem("name");
console.log(name);
removeItem() → Remove One Item
JavaScript 
localStorage.removeItem("name");
clear() → Remove All
JavaScript 
localStorage.clear();

Important Rule

LocalStorage stores only STRINGS.

If you store array or object directly → it will break.

So we must use:

  • JSON.stringify()
  • JSON.parse()

Storing Object in LocalStorage

JavaScript 
let user = {
  name: "Sanjay",
  age: 22
};

localStorage.setItem("user", JSON.stringify(user));

Getting Object Back

JavaScript 
let storedUser = JSON.parse(localStorage.getItem("user"));
console.log(storedUser.name);
