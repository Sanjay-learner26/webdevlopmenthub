IMAGES & MULTIMEDIA

Images and multimedia make webpages attractive and interactive. HTML allows us to add images, audio, videos, and external content. These elements improve user experience.


Tag: <img>

Tag Explanation

The <img> tag is used to display images. It requires src (image path) and alt (alternative text). It is a self-closing tag.

Code Example: 
<img src="photo.jpg" alt="Sample Image">

Tag: <audio>

Tag Explanation

The <audio> tag adds sound to a webpage. The controls attribute adds play/pause buttons. Audio files are added using <source>.

Code Example: 
<audio controls>
   <source src="audio.mp3" type="audio/mpeg">
</audio>

Tag: <video>

Tag Explanation

The <video> tag displays videos. The controls attribute shows video controls. Supports multiple formats using <source>.

Code Example: 
<video width="300" controls>
   <source src="video.mp4" type="video/mp4">
</video>

Tag: <iframe>

Tag Explanation

The <iframe> tag embeds another webpage inside your page. Commonly used to embed YouTube videos or maps. It requires a src attribute.

Code Example: 
<iframe width="300" height="200"
src="https://www.youtube.com/embed/dQw4w9WgXcQ">
</iframe>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Images and Multimedia</title>
</head>
<body>

   <h1>Media Example</h1>

   <h2>Image</h2>
   <img src="photo.jpg" alt="Profile Photo" width="200">

   <h2>Audio</h2>
   <audio controls>
      <source src="audio.mp3" type="audio/mpeg">
   </audio>

   <h2>Video</h2>
   <video width="300" controls>
      <source src="video.mp4" type="video/mp4">
   </video>

   <h2>YouTube Video</h2>
   <iframe width="300" height="200"
   src="https://www.youtube.com/embed/dQw4w9WgXcQ">
   </iframe>

</body>
</html>

Output Explanation

The page shows:
  • An image
  • An audio player
  • A video player
  • An embedded YouTube video
