Text Properties

Text properties are used to control the appearance of text like color, size, alignment, spacing, decoration, etc.


  1. color

Changes the text color

Example <p style="color: blue;">This text is blue.</p>
You can use:
Color name → red
Hex → #ff0000
RGB → rgb(255,0,0)

  1. text-align

Aligns text horizontally.

Example <p style="text-align: center;">This text is centered.</p>
Values
  • left
  • Right
  • Center
  • Justify

  1. text-decoration

Adds underline, overline, line-through.

Example <p style="text-decoration: underline;">Underlined text</p> <p style="text-decoration: line-through;">Striked text</p>

  1. text-transform

Changes letter case.

<p style="text-transform: uppercase;">hello world</p> <p style="text-transform: lowercase;">HELLO WORLD</p> <p style="text-transform: capitalize;">hello world</p>

  1. font-size

Changes text size.

<p style="font-size: 20px;">Big text</p>
Other units
  • px
  • em
  • rem
  • %

  1. font-family

Changes font style.

This is monospace text.


  1. font-weight

Controls thickness of text

<p style="font-weight: bold;">Bold text</p> <p style="font-weight: 300;">Light text</p>

  1. letter-spacing

Space between letters.

<p style="letter-spacing: 3px;">Spaced Letters</p>

  1. line-height

Controls space between lines.

<p style="line-height: 30px;"> This is line one.
This is line two. </p>

  1. text-shadow

Adds shadow to text.

<p style="text-shadow: 2px 2px 5px gray;"> Shadow Text </p>
  1. Background Properties

Explain what it does

Example <p style="background-color: yellow;"> This has yellow background </p>
Mention values:
color name
hex
rgb

  1. background-image

Used to add image as background

Example 
p {
  background-image: url("image.jpg");
}

  1. background-repeat
Explain values:
repeat (default)
no-repeat
repeat-x
repeat-y
Example 
background-repeat: no-repeat;

  1. background-position

Controls image position.

Values:
left
right
center
top
bottom
Example 
background-position: center;

  1. background-size

Very important.

Values:
cover
contain
px values
Example 
background-size: cover;

  1. background-attachment
Explain:
scroll (default)
fixed
Example 
background-attachment: fixed;

  1. background (Shorthand Property)

Very important (intermediate level)

Example 
background: yellow url("image.jpg") no-repeat center cover;

Explain that this combines all background properties.
