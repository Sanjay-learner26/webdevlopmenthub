INTRODUCTION TO HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content like headings, paragraphs, images, and links. HTML works together with CSS (design) and JavaScript (functionality).


Tag: <p>

Tag Explanation

The <p> tag defines a paragraph. Browsers automatically add space before and after paragraphs. It is a block-level element.

Code Example: 
<p>This is a paragraph in HTML.</p>

Tag: <h1> </h1>

Tag Explanation

The <h1> tag defines the most important heading. It is usually used as the main title of the page. Search engines give high importance to <h1>.

Code Example: 
<h1>Welcome to HTML</h1>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
    <title>Introduction Example</title>
</head>
<body>

    <h1>Welcome to My Website</h1>
    <p>HTML is used to create web pages.</p>
    <p>It structures content properly.</p>

</body>
</html>

Output Explanation

The browser displays:
  • A large heading: "Welcome to My Website"
  • Two paragraphs below it
  • The browser tab shows "Introduction Example"
