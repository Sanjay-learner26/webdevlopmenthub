ARRAYS

What is an Array?

An array is a special variable that can store multiple values in a single variable.

Instead of:
Example: 
            let name1 = "Sanjay";
let name2 = "Ravi";
let name3 = "Kumar";
We use:
Example: 
            let names = ["Sanjay", "Ravi", "Kumar"];

Creating Arrays

Example: 
            let numbers = [10, 20, 30];
let mixed = ["Sanjay", 22, true];

Accessing Array Elements

Arrays use index (starts from 0)

Example: 
        let colors = ["red", "blue", "green"];

console.log(colors[0]); // red
console.log(colors[1]); // blue

Changing Values

Example: 
            colors[1] = "yellow";
console.log(colors);

length Property

Example: 
            let arr = [1,2,3,4];
console.log(arr.length); // 4

Looping Through Arrays

Using for loop
Example: 
            let nums = [10,20,30];

for(let i = 0; i < nums.length; i++){
  console.log(nums[i]);
}

Using for...of

Example: 
            for(let value of nums){
  console.log(value);
}

Adding & Removing Elements

push() → Add at end
Example: 
            let arr = [1,2];
arr.push(3);
pop() → Remove last
Example: 
            arr.pop();
unshift() → Add at start
Example: 
            arr.unshift(0);
shift() → Remove first
Example: 
            arr.shift();
Advanced Important Methods
map()

Creates new array.

Example: 
                let nums = [1,2,3];

let doubled = nums.map(function(num){
  return num * 2;
});

console.log(doubled); // [2,4,6]

Used when transforming data.

filter()

Filters values.

Example: 
                let nums = [10, 25, 30, 15];

let result = nums.filter(function(num){
  return num > 20;
});

console.log(result); // [25,30]

Used in search, filtering products.

reduce()

Reduces to single value

Example: 
                let nums = [1,2,3,4];

let sum = nums.reduce(function(total, num){
  return total + num;
}, 0);

console.log(sum); // 10

Used in cart total calculation.

find()

Returns first matching element.

Example: 
                let users = [
  {id:1, name:"Sanjay"},
  {id:2, name:"Ravi"}
];

let user = users.find(function(u){
  return u.id === 2;
});

console.log(user);
includes()
Example: 
                let fruits = ["apple","banana"];

console.log(fruits.includes("banana")); // true
Array of Objects

This is real project structure.

Example: 
                let products = [
  {id:1, name:"Shirt", price:500},
  {id:2, name:"Shoes", price:1000}
];
Accessing:
Example: 
                console.log(products[0].name);
