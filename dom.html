DOM (Document Object Model)

This is where real web development starts.

What is DOM?

DOM stands for DOM

When a browser loads your HTML page:
  • It converts HTML into a JavaScript object structure
  • Each HTML element becomes an object.
  • JavaScript can access and modify those objects
Example: 
<h1 id="title">Hello</h1>

Browser converts it into a DOM object. JavaScript can now change it.

Why DOM is Important?

DOM allows you to:
  • Change text
  • Add elements
  • Remove elements
  • Handle user input
Without DOM → JavaScript cannot change HTML

Selecting Elements

Before changing something, we must select it.

getElementById()
HTML 
<h1 id="title">Hello</h1>
JavaScript 
let heading = document.getElementById("title");
console.log(heading);
getElementsByClassName()
HTML 
<p class="text">One</p>
<p class="text">Two</p>
JavaScript 
let elements = document.getElementsByClassName("text");
console.log(elements);

Returns HTMLCollection

querySelector() (Very Important)

Selects first matching element.

JavaScript 
let element = document.querySelector("#title");
let element2 = document.querySelector(".text");
querySelectorAll() (Very Important)

Selects all matching elements.

JavaScript 
let elements = document.querySelectorAll(".text");

Returns NodeList

Changing Content innerText

Changes text only.

JavaScript 
heading.innerText = "Welcome Sanjay";
innerHTML

Changes HTML content.

JavaScript 
heading.innerHTML = "<span style='color:red'>Hello</span>";
innerHTML can insert HTML tags.

Changing Styles

JavaScript 
heading.style.color = "blue";
heading.style.backgroundColor = "yellow";
Better way: Use class.
JavaScript 
heading.classList.add("active");

Working with Classes

JavaScript 
element.classList.add("box");
element.classList.remove("box");
element.classList.toggle("box");
toggle → adds if not present, removes if present.

Creating Elements Dynamically

JavaScript 
let newPara = document.createElement("p");
newPara.innerText = "New Paragraph";

document.body.appendChild(newPara);
Used in:
  • Adding products
  • Adding list items

Removing Elements

JavaScript 
newPara.remove();
Event Handling (Very Important)

DOM becomes powerful with events.

Click Event

HTML 
<button id="btn">Click Me</button>
JavaScript 
let btn = document.getElementById("btn");

btn.addEventListener("click", function(){
  alert("Button Clicked!");
});

Input Event

HTML 
<input type="text" id="name">
JavaScript 
let input = document.getElementById("name");

input.addEventListener("input", function(){
  console.log(input.value);
});
