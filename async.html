Async JavaScript

What is Asynchronous JavaScript?

JavaScript normally runs line by line. But async tasks run in background.

Example:

console.log("Start");

setTimeout(function(){
  console.log("After 2 seconds");
}, 2000);

console.log("End");

Promises

let promise = new Promise(function(resolve, reject){
  resolve("Success");
});

async / await

  return "Data Loaded";
}
