OPERATORS

JavaScript operators are special symbols or keywords that perform operations on one or more values (operands) and return a result. They are essential for performing calculations, comparing data, and controlling program flow.

JavaScript includes several categories of operators for different purposes:
Arithmetic Operators
symbols 
+  -  *  /  %  **
example 
let a = 10;
let b = 5;

console.log(a + b); // 15

Comparison Operators

example 
==   ===   !=   !==   >   <
Important
Example 
5 == "5"   // true
5 === "5"  // false

Always use ===

Logical Operators

symbols 
&&   ||   !
Example 
let age = 20;

if(age > 18 && age < 30){
  console.log("Young adult");
}
