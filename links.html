LINKS & NAVIGATION

Links are used to connect webpages. The <a> tag creates hyperlinks. Navigation makes websites user-friendly.


Tag: <a>

Tag Explanation

The <a> tag creates hyperlinks. It uses the href attribute to define the destination. Users can click the link to move to another page.

Code Example: 
<a href="about.html">Go to About Page</a>

Attribute: target

Tag Explanation

The target="_blank" attribute opens the link in a new tab. It improves user navigation.

Code Example: 
<a href="https://google.com" target="_blank">Open Google</a>

Mailto Link

Tag Explanation

Used to send emails directly. Opens the default email application.

Code Example: 
<a href="mailto:test@gmail.com">Send Email</a>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Links Example</title>
</head>
<body>

   <h1>Link Examples</h1>

   <a href="topic5-lists.html">Go to Lists Page</a>
   <br><br>

   <a href="https://www.google.com" target="_blank">
      Open Google in New Tab
   </a>
   <br><br>

   <a href="mailto:example@gmail.com">Send Email</a>

</body>
</html>

Output Explanation

The page shows:
  • Internal link to another topic page
  • External link opening in a new tab
  • Email link that opens mail application
