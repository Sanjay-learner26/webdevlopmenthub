OBJECTS

What is an Object?

An object is a data structure used to store related data in key–value pairs.

Person → has name, age, city
Car → has brand, model, price

So, we group related data inside one structure.

Real life example:
Instead of: 
let name = "Sanjay";
let age = 22;
let city = "Chennai";
We use: 
let person = {
  name: "Sanjay",
  age: 22,
  city: "Chennai"
};
Here:
  • name → key
  • "Sanjay" → value
Why Objects Are Important?
  • Used to represent real-world entities
  • Used in APIs
  • Used in databases
  • Used in JSON
In real web projects, 80% of data is objects.
Example: 
{
  id: 1,
  title: "Product",
  price: 500
}

Creating Objects

Method 1: Object Literal (Most Common)
Example 
let student = {
  name: "Sanjay",
  age: 22,
  isStudent: true
};
Method 2: Using new Object()
Example: 
let student = new Object();
student.name = "Sanjay";
student.age = 22;
Not commonly used.

Accessing Object Properties

Dot Notation (Recommended)
Example: 
console.log(student.name);
Bracket Notation
Example: 
console.log(student["age"]);
When to use bracket?
  • When key is dynamic
  • When key has spaces
Example: 
let user = {
  "full name": "Sanjay S"
};

console.log(user["full name"]);
Adding New Properties
Example: 
student.city = "Chennai";
Updating Properties
Example: 
student.age = 23;
Deleting Properties
Example: 
delete student.isStudent;
Object Methods
Example: 
let person = {
  name: "Sanjay",
  greet: function(){
    console.log("Hello " + this.name);
  }
};

person.greet();
this Keyword

this refers to the current object.

Example: 
this.name

The name inside this object.

Important:

this behaves differently in arrow functions.

Looping Through Objects
Using for...in
Example: 
for(let key in student){
  console.log(key + " : " + student[key]);
}
Object Destructuring (ES6)

Extract values easily.

Example: 
let person = {
  name: "Sanjay",
  age: 22
};

let {name, age} = person;

console.log(name);
Why use?
  • Used heavily in React

Spread Operator (...)

Used to copy objects

Example: 
let person = {
  name: "Sanjay",
  age: 22
};

let newPerson = {...person};
Used in:
  • Updating state
  • Merging objects
Object Inside Array

Real project structure:

Example: 
let users = [
  {id:1, name:"Sanjay"},
  {id:2, name:"Ravi"}
];
Access
Example: 
console.log(users[0].name);
Used in:
  • Product listing
  • User list
  • Cart system
