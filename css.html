CSS

CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance and layout of HTML web pages. It is used to design elements such as colors, fonts, spacing, alignment, and positioning.

CSS works together with HTML:

  • HTML → Creates the structure of a webpage

  • CSS → Designs and styles the webpage

What Does “Cascading” Mean?

The word Cascading means that styles are applied based on priority.

If multiple styles apply to the same element:

  • Inline CSS → Highest priority

  • Internal CSS → Medium priority

  • External CSS → Lower priority

The browser decides which style to apply based on this order.


Advantages of CSS

  • Makes websites beautiful
  • Saves time (one CSS file can style multiple pages)
  • Separates design from structure
  • Makes websites responsive
  • Improves user experience

CSS Syntax Structure

CSS has two main parts:
Syntax 
selector {
   property: value;
}

Example 
p {
   color: blue;
}

  • p → Selector

  • color → Property

  • blue → Value

How CSS Works

  • Browser reads HTML

  • Browser reads CSS

  • CSS styles are applied to matching HTML elements

  • Page is displayed with design
