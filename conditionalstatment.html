CONDITIONAL STATEMENTS

Conditional statements in JavaScript control the flow of execution, allowing the program to perform different actions based on whether a specified condition is true or false. These conditions are typically evaluated using comparison and logical operators.

The main types of conditional statements in JavaScript are:
if Statement

In JavaScript, the if statement executes a block of code if a specified condition is true.

Example 
if(age > 18){
  console.log("Adult");
}
if...else
Example 
if(age > 18){
  console.log("Adult");
} else {
  console.log("Minor");
}

else if

Example 
if(age < 13){
  console.log("Child");
}
else if(age < 18){
  console.log("Teen");
}
else {
  console.log("Adult");
}

Switch

Example 
let day = "Monday";

switch(day){
  case "Monday":
    console.log("Start week");
    break;
}

Loops

JavaScript provides several types of loops to repeatedly execute code.

For Loop
Syntax 
for(initialization; condition; increment){
  // code
}
  • initialization → starting point
  • condition → loop runs while condition is true
  • increment → increases value each time
Example 
for(let i = 1; i <= 5; i++){
  console.log(i);
}

Output:
1 2 3 4 5
while Loop
Syntax 
while(condition){
   // code
}
Example 
let i = 1;

while(i <= 5){
  console.log(i);
  i++;
}
do...while Loop
Syntax 
do{
   // code
} while(condition);
Example 
let i = 1;

do{
  console.log(i);
  i++;
} while(i <= 5);
for...of Loop
Example 
let colors = ["red", "blue", "green"];

for(let color of colors){
  console.log(color);
}
for...in Loop
Example 
let person = {
  name: "Sanjay",
  age: 22
};

for(let key in person){
  console.log(key + " : " + person[key]);
}
break Statement
Example 
for(let i = 1; i <= 10; i++){
  if(i === 5){
    break;
  }
  console.log(i);
}

Output:
1 2 3 4
continue Statement
Example 
for(let i = 1; i <= 5; i++){
  if(i === 3){
    continue;
  }
  console.log(i);
}

Output:
1 2 4 5
Nested Loops
Example 
for(let i = 1; i <= 3; i++){
  for(let j = 1; j <= 2; j++){
    console.log(i, j);
  }
}
