CSS Box Model

What is CSS Box Model?

Every HTML element is treated as a box in CSS. The CSS Box Model describes how space is handled around elements.

4 Parts of the Box Model
  1. Content
  2. Padding
  3. Border
  4. Margin

  1. Content

This is the actual text or image inside the element.

Example <p>Hello World</p>

The words "Hello World" are the content.

You control it using:

  • width
  • height

  1. Padding

Padding is the space inside the box, between content and border.

Example 
p {
  padding: 20px;
}

This adds space inside the paragraph


  1. Border

Border wraps around padding and content.

Example 
p {
  border: 2px solid black;
}
You can control:
border-width
border-style
border-color

  1. Margin

Margin is the space outside the border. It creates distance between elements.

Example 
p {
  margin: 30px;
}
