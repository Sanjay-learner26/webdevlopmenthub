BLOCK & INLINE ELEMENTS

HTML elements are divided into block-level and inline elements. Block elements take full width and start on a new line. Inline elements only take necessary width and stay in the same line.


Tag: <div> (Block Element)

Tag Explanation

The <div> tag is a block-level container. It is used to group elements together. It is mainly used for layout and structure.

Code Example: 
<div>
   <p>This paragraph is inside a div.</p>
</div>

Tag: <span> (Inline Element)

Tag Explanation

The <span> tag is an inline container. It is used to style small portions of text. It does not start on a new line.

Code Example: 
<p>This is <span>red</span> text.</p>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Block and Inline Elements</title>
</head>
<body>

   <h1>Block vs Inline</h1>

   <div>
       <p>This is inside a div block element.</p>
   </div>

   <p>This is <span>blue</span> word inside a sentence.</p>

</body>
</html>

Output Explanation

The page shows:
  • A heading
  • A block section using div
  • A sentence with inline span usage
  • Block elements start new lines, inline elements stay in line
