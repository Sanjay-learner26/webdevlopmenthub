TABLES

Tables are used to display data in rows and columns. They are useful for structured information like marks, prices, and schedules. Tables contain rows (<tr>) and cells (<td> or <th>).


Tag: <table>

Tag Explanation

Defines the table container. All table content must be inside this tag.

Code Example: 
<table border="1">
</table>

Tag: <tr>

Tag Explanation

Defines a table row. Each row contains cells.

Code Example: 
<tr>
</tr>

Tags: <th> and <td>

Tag Explanation

<th> defines header cells (bold and centered). <td> defines normal data cells.

Code Example: 
<tr>
   <th>Name</th>
   <th>Age</th>
</tr>

Attributes: rowspan & colspan

Tag Explanation

rowspan merges rows. colspan merges columns.

Code Example: 
<td colspan="2">Merged Cell</td>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Table Example</title>
</head>
<body>

   <h1>Student Table</h1>

   <table border="1">
      <tr>
         <th>Name</th>
         <th>Age</th>
      </tr>

      <tr>
         <td>Sanjay</td>
         <td>22</td>
      </tr>

      <tr>
         <td colspan="2">End of Data</td>
      </tr>
   </table>

</body>
</html>

Output Explanation

The page shows:
  • A table with two columns
  • Student name and age
  • Last row merges two columns
