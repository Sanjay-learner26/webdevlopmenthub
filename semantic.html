SEMANTIC HTML

Semantic HTML tags clearly describe the meaning of content. They improve readability, SEO, and accessibility. Instead of using only <div>, semantic tags define proper sections of a webpage.


Tag: <header>

Tag Explanation

The <header> tag represents introductory content. It usually contains logo, title, or navigation links. It appears at the top of a page or section.

Code Example: 
<header>
   <h1>My Website</h1>
</header>

Tag: <nav>

Tag Explanation

The <nav> tag defines navigation links. It is used for menus and important links. Improves SEO structure.

Code Example: 
<nav>
   <a href="#">Home</a>
   <a href="#">About</a>
</nav>

Tag: <section>

Tag Explanation

The <section> tag groups related content. Each section usually has a heading. Used to organize page content.

Code Example: 
<section>
   <h2>About Us</h2>
   <p>We create websites.</p>
</section>

Tag: <article>

Tag Explanation

The <article> tag represents independent content. Used for blog posts, news articles, or posts. It can stand alone.

Code Example: 
<article>
   <h2>Blog Post</h2>
   <p>This is an article.</p>
</article>

Tag: <footer>

Tag Explanation

The <footer> tag defines the bottom section. It usually contains copyright or contact information.

Code Example: 
<footer>
   <p>© 2026 My Website</p>
</footer>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Semantic HTML</title>
</head>
<body>

   <header>
      <h1>My Blog</h1>
   </header>

   <nav>
      <a href="#">Home</a> |
      <a href="#">Posts</a>
   </nav>

   <section>
      <article>
         <h2>First Article</h2>
         <p>This is a blog post example.</p>
      </article>
   </section>

   <footer>
      <p>© 2026 My Blog</p>
   </footer>

</body>
</html>

Output Explanation

The page shows:
  • Header with blog title
  • Navigation links
  • Article section
  • Footer at bottom
