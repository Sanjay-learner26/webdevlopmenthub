FORMS

Forms collect user input such as names, emails, and passwords. They are essential for login pages, registration forms, and surveys. Form data can be sent to a server using action and method.


Tag: <form>

Tag Explanation

The <form> tag wraps all input elements. It sends data to the server.

Code Example: 
<form action="#" method="post">
</form>

Tag: <input>

Tag Explanation

The <input> tag creates input fields. Different type attributes create different fields.

Code Example: 
<input type="text" placeholder="Enter Name">
<input type="email" required>
<input type="password">
<input type="submit">

Tag: <label>

Tag Explanation

The <label> tag describes input fields. Improves accessibility and usability.

Code Example: 
<label>Name:</label>
<input type="text">

Tag: <textarea>

Tag Explanation

Creates a multi-line text box. Used for comments or messages.

Code Example: 
<textarea rows="4" cols="30"></textarea>

Tags: <select> and <option>

Tag Explanation

Creates a dropdown list. Options are defined using <option>.

Code Example: 
<select>
   <option>HTML</option>
   <option>CSS</option>
</select>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Form Example</title>
</head>
<body>

   <h1>Registration Form</h1>

   <form action="#" method="post">

      <label>Name:</label><br>
      <input type="text" required><br><br>

      <label>Email:</label><br>
      <input type="email" required><br><br>

      <label>Password:</label><br>
      <input type="password"><br><br>

      <label>Course:</label><br>
      <select>
         <option>HTML</option>
         <option>CSS</option>
      </select><br><br>

      <label>Message:</label><br>
      <textarea rows="4" cols="30"></textarea><br><br>

      <input type="submit" value="Register">

   </form>

</body>
</html>

Output Explanation

The page shows:
  • Input fields for name, email, and password
  • Dropdown menu
  • Textarea
  • Submit button
  • Required validation for some fields
