HTML ENTITIES

HTML entities are special codes used to display reserved characters. They begin with & and end with ;. Used for symbols like <, >, and copyright.


Entity: < and >

Tag Explanation

Used to display < and > symbols. Prevents browser interpretation problems.

Code Example: 
<p>5 &lt; 10</p>

Entity: &copy;

Tag Explanation

Displays copyright symbol.

Code Example: 
<p>&copy; 2026 My Company</p>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Entities Example</title>
</head>
<body>

   <p>5 &lt; 10</p>
   <p>&copy; 2026 My Website</p>

</body>
</html>

Output Explanation

  • Displays "5 < 10"
  • Shows copyright symbol
