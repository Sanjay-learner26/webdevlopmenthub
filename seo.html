SEO Basics in HTML

SEO (Search Engine Optimization) helps websites rank higher in search engines like Google. HTML provides special <meta> tags to improve search visibility. Good SEO structure increases website traffic and user reach.


Tag: <meta name="description">

Tag Explanation

The meta description gives a short summary of the webpage. It appears in search engine results below the title. It should be clear, short, and relevant.

Code Example: 
<head>
  <meta name="description" content="Learn HTML from beginner to advanced with examples.">
</head>

Tag: <meta name="keywords">

Tag Explanation

The meta keywords tag lists important keywords of the webpage. It helps search engines understand page content. Modern search engines give it less importance but it is useful for learning.

Code Example: 
<head>
  <meta name="keywords" content="HTML, CSS, Web Development, Beginner HTML">
</head>

Tag: <meta name="viewport">

Tag Explanation

The viewport meta tag makes the website responsive. It controls layout on mobile devices. It is very important for modern websites.

Code Example: 
<head>
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>

   <title>SEO Example</title>

   <meta name="description" content="Learn HTML SEO Basics">
   <meta name="keywords" content="HTML, SEO, Web Development">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">

</head>

<body>

   <h1>SEO Learning Page</h1>
   <p>This page demonstrates SEO meta tags.</p>

</body>
</html>

Output Explanation

  • Meta description helps search results
  • Meta keywords help content understanding
  • Viewport helps mobile responsiveness
