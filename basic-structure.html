BASIC STRUCTURE TAGS

Basic structure tags create the skeleton of a web page. Every HTML file must contain these essential tags. Without them, browsers cannot properly render the page.


Tag: <!DOCTYPE html>

Tag Explanation

Declares the document type as HTML5. It must always be the first line of the document.

Code Example: 
<!DOCTYPE html>

Tag: <html>

Tag Explanation

The root element of the web page. All other HTML elements must be inside this tag.

Code Example: 
<html> </html>

Tag: <head>

Tag Explanation

Contains metadata about the web page. It does not display visible content.

Code Example: 
<head>
   <title>My Website</title>
</head>

Tag: <body>

Tag Explanation

Contains all visible content. Everything users see is inside this tag.

Code Example: 
<body>
   <h1>Hello World</h1>
</body>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Basic Structure</title>
</head>
<body>

   <h1>My First Page</h1>
   <p>This page shows basic structure tags.</p>

</body>
</html>

Output Explanation

Displays:
  • Heading: "My First Page"
  • A paragraph below it
  • Tab title: "Basic Structure"
