Data Types in JavaScript

Data type defines what kind of value a variable can store. JavaScript supports primitive and non-primitive data types.

Primitive Data Types

1. String

Used to store text values.

Example 
let name = "Sanjay";
let message = 'Hello';

Can be written using:

  • " "
  • ' '
  • ` ` (template literal)

2. Number

Used to store integers and decimal values.

Example 
let age = 22;
let price = 99.5;

3. Boolean

Boolean stores only two values.

  • true
  • false
Example 
let isStudent = true;
let isLoggedIn = false;

4. Undefined

Variable declared but value not assigned.

Example 
let x;
console.log(x); // undefined

5. Null

Represents empty or unknown value.

Example 
let data = null;

Non-Primitive Data Types

Array

Stores multiple values in one variable.

Example 
let fruits = ["apple", "banana", "orange"];

console.log(fruits[0]);

Object

Stores data in key-value pairs.

Example 
let person = {
  name: "Sanjay",
  age: 22
};

console.log(person.name);

Type Checking

Use typeof operator.

Example 
let name = "Sanjay";
console.log(typeof name);
