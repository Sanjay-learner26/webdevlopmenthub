TEXT & FORMATTING TAGS

Text formatting tags help style and emphasize text. They make content bold, italic, marked, or structured. These tags improve readability and presentation.


Tag: <strong>

Tag Explanation

Makes text bold and important. Used for meaningful emphasis.

Code Example: 
<strong>Important Notice</strong>

Tag: <em>

Tag Explanation

Emphasizes text. Usually displayed in italic style.

Code Example: 
<em>This is emphasized text</em>

Tag: <br>

Tag Explanation

Creates a line break. Does not require a closing tag.

Code Example: 
Hello<br>World

Tag: <hr>

Tag Explanation

Creates a horizontal line. Used to separate sections.

Code Example: 
<hr>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Text Formatting</title>
</head>
<body>

   <h1>Formatting Example</h1>

   <p>This is <strong>important</strong> text.</p>
   <p>This is <em>emphasized</em> text.</p>

   Hello<br>World

   <hr>

   <p>Section separated by line.</p>

</body>
</html>

Output Explanation

The page shows:
  • A heading
  • Bold text for "important"
  • Italic text for emphasized text
  • Line break between Hello and World
  • A horizontal line separator
