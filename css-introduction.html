What is Css ?

CSS is a styling language used to design and style web pages.

  • CSS stands for Cascading Style Sheet.
  • CSS describes how HTML elements are displayed on screen, paper, or other media.
  • CSS saves time by controlling layout of multiple pages.
  • External stylesheets are stored in CSS files.

Why Css is used?

Types of CSS
  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS

Inline CSS means writing CSS directly inside an HTML tag using the style attribute.

Syntax 
<tag style="property: value;">Content</tag>

Example 
<p style="color:red;">This is red text</p>

Internal CSS

Internal CSS is written inside the same HTML file using the <style> tag inside the <head> section.

Syntax 
<head>
<style>
selector {
    property: value;
}
</style>
</head>

Example 
<!DOCTYPE html>
<html>
<head>
<title>Internal CSS Example</title>

<style>
h1 {
    color: blue;
}

p {
    font-size: 18px;
}
</style>

</head>

<body>
<h1>Hello World</h1>
<p>This is styled using internal CSS.</p>
</body>
</html>

External CSS

External CSS is written in a separate .css file and linked using the <link> tag.

Used when:
You have multiple pages.
You want clean structure.
You want reusable styles.
Syntax for HTML file 
<head>
<link rel="stylesheet" href="style.css">
</head>

Syntax for CSS file 
selector {
    property: value;
}
