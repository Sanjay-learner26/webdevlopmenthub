LISTS

Lists are used to group related items. HTML provides ordered, unordered, and description lists. They help structure content clearly.


Tag: <ul> (Unordered List)

Tag Explanation

Creates a bullet list. Items must be written inside <li> tags.

Code Example: 
<ul>
   <li>HTML</li>
   <li>CSS</li>
</ul>

Tag: <ol> (Ordered List)

Tag Explanation

Creates a numbered list. Useful for steps or ranking.

Code Example: 
<ol>
   <li>Step 1</li>
   <li>Step 2</li>
</ol>

Tag: <dl> (Description List)

Tag Explanation

Used to define terms and descriptions. Contains <dt> (term) and <dd> (description).

Code Example: 
<dl>
   <dt>HTML</dt>
   <dd>Markup language for webpages</dd>
</dl>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Lists Example</title>
</head>
<body>

   <h1>List Types</h1>

   <h2>Unordered List</h2>
   <ul>
       <li>Apple</li>
       <li>Banana</li>
   </ul>

   <h2>Ordered List</h2>
   <ol>
       <li>Login</li>
       <li>Register</li>
   </ol>

   <h2>Description List</h2>
   <dl>
       <dt>HTML</dt>
       <dd>Structure of webpage</dd>
   </dl>

</body>
</html>

Output Explanation

The page shows:
  • A bullet list (Apple, Banana)
  • A numbered list (Login, Register)
  • A description list with term and explanation
