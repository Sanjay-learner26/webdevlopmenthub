ATTRIBUTES

Attributes provide additional information about HTML elements. They are written inside opening tags. Attributes usually have name and value.


Attribute: id

Tag Explanation

The id attribute gives a unique identifier. It must not be repeated in the same page. Used in CSS and JavaScript.

Code Example: 
<p id="intro">Welcome</p>

Attribute: class

Tag Explanation

The class attribute groups elements together. Multiple elements can share the same class.

Code Example: 
<p class="text">Hello</p>

Attribute: title

Tag Explanation

The title attribute shows tooltip text. It appears when hovering over an element.

Code Example: 
<p title="This is tooltip">Hover me</p>

Full Combined Example

HTML 
<!DOCTYPE html>
<html>
<head>
   <title>Attributes Example</title>
</head>
<body>

   <h1 id="main-heading">Attributes Demo</h1>

   <p class="text">This paragraph has a class.</p>

   <p title="Tooltip text">Hover over this text.</p>

</body>
</html>

Output Explanation

The page shows:
  • Heading with id attribute
  • Paragraph with class attribute
  • Tooltip text when hovering
