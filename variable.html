What is a Variable?

A variable is a container used to store data. It allows us to store values like numbers, strings, or objects and use them later.

Think like

  • Variable = Box
  • Value = Item inside the box

Why We Need Variables?

  • To store user input
  • To store calculations
  • To reuse data
  • To update values dynamically
Example 
let username = "Sanjay";

Types of Variable Declarations

var (Old method)

Example 
var age = 20;

Content:

  • Function scoped
  • Can be redeclared
  • Can be updated
  • Hoisted
Not recommended in modern JavaScript.

let (Recommended)

Example: 
let age = 22;

Content:

  • Block scoped
  • Cannot redeclare
  • Can update value
Example: 
let score = 10;
score = 20;   // allowed

const (Constant)

Example: 
const pi = 3.14;

Content:

  • Block scoped
  • Cannot redeclare
  • Cannot update
Example: 
const age = 22;
age = 25; // ❌ Error
Important:

If value should not change → use const

Variable Naming Rules

Allowed:
  • Letters
  • _
  • $
  • Numbers (not at start)
Example: 
let userName;
let _count;
let $price;
Not Allowed:
Example: 
let 1name;   // error
let var;     // reserved keyword
Best Practices:
  • Use meaningful names
  • Use camelCase
  • Prefer const > let
Example: 
let totalPrice = 500;

DATA TYPES

What is Data Type?

Data type defines what kind of value a variable holds.

Example: 
let name = "Sanjay";   // String
let age = 22;          // Number

Primitive Data Types

String

Stores text.

Example 
let message = "Hello";
Can use:
  • ""
  • ''
  • `` (backticks)
Example: 
let name = "Sanjay";
console.log(`Hello ${name}`);
Number

Stores integers and decimals.

Example: 
let num1 = 10;
let num2 = 10.5;
Boolean

let isLoggedIn = true;

Only:
  • True
  • False

Undefined

Example 
let x;
console.log(x); // undefined

Null

Example: 
let data = null;

Represents empty value.

Non-Primitive Data Types

Array
Example: 
let colors = ["red", "blue"];
Object
Example: 
let person = {
  name: "Sanjay",
  age: 22
};
