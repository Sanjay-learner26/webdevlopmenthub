BEGINNER

For someone who is just starting web development

What HTML is?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc

Basic tags

Tags Descriptions
<DOCTYPE html> Defines a Document
<html> Defines a HTML Document
<head> Contains meta data / information about a Document
<title> Defines a Title of a Document
<body> Defines a Document Body
<h1> - <h6> Defines HTML Headings
<p> Defines a paragraph
<br> Defines a break
<hr> Defines a thematic change in the content
<!--...--> Defines a comment

INTERMEDIATE

For learners who know HTML basics and are ready to style pages with CSS and build layouts.

Introduction

At the intermediate level, you will learn how to style web pages, create layouts, and make your websites visually appealing using CSS.

Topics covered

  1. What CSS is
  2. Inline, Internal, External CSS
  3. Colors, Fonts, Text styles
  4. Box model: padding, margin, border
  5. Simple layout: div, section, header, footer
  6. Lists and tables styling
  7. Linking CSS to HTML
  8. Basic responsive design (optional, just concept)
