CSS Selectors

CSS selectors are used to select (target) HTML elements so we can apply styles to them.

Main Types of CSS Selectors

  1. Element Selector

Targets all elements of that type.

HTML Example 
<p>Hello World</p>

CSS 
p {
  color: blue;
}

<p> tags become blue


ID Selector

Used for styling one unique element

HTML Example 
<p id="title">Hello World</p>

CSS 
#title {
  color: red;
}

ID should be used only once per page.


Class Selector (.)

Used for styling multiple elements

HTML Example 
<p class="text">First Paragraph</p>
<p class="text">Second Paragraph</p>

CSS 
.text {
  color: green;
}

Both paragraphs turn green.


Universal Selector (*)

Targets everything

CSS 
* {
  margin: 0;
}

Used to remove default spacing.


Group Selector

Style multiple elements together.

CSS 
h1, h2, p {
  color: purple;
}
